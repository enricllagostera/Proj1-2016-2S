Dicas para manual de jogo analógico

Esta é apenas uma referência. Alguns jogos terão mais ou menos detalhes em cada seção ou mesmo outras seções específicas (por exemplo, uma seção explicando combate entre jogadores em mais detalhe). Fique atento às necessidades de seu jogo.

Estrutura básica

  1. Breve resumo: número de jogadores, contexto, objetivos resumidos, papéis dos jogadores
  2. Materiais e preparação necessária para uma partida
  3. Objetivos e condições de vitória
  4. Fluxo do jogo: rodadas, como é a vez de cada jogador
  5. Casos especiais e regras específicas
  6. Como o jogo termina?

Dicas para escrever boas regras

  • Use a narrativa / tema / contexto para explicar o jogo.
  • Deixe claro toda a preparação necessária.
  • Comece pelos objetivos e condições de vitória / derrota.
  • Tente sempre ir do que é mais geral para o que é mais específico.
  • Explique primeiro o que os jogadores podem fazer e depois o que não podem.
  • Evitar novos termos ou jargão: se for explicar um termo do jogo, faça-o na primeira vez que é mencionado e depois disso use sempre o mesmo termo para aquilo.
  • Leia em voz alta para ver se estão claras: é assim que seus jogadores vão fazer.
  • Organize o texto em partes para facilitar a leitura e referência.
  • Use diagramas ou ilustrações / fotos para facilitar o entendimento quando necessário.
  • Teste suas regras com alguém que está fora do grupo.
  • Use exemplos sucintos para tornar a explicação mais concreta.
  • Seja amigável, respeitoso e se refira diretamente aos jogadores.
  • Cuide para que o layout, tipografia e cores sejam legíveis e facilitem a compreensão dos jogadores.
